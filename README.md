# 🌸 Iris Flower Prediction System

This project is a Machine Learning-based web application that predicts the species of an iris flower using both numerical features and image input.

---

## 🚀 Features

- 📊 **Feature-Based Prediction**
  - Input: Sepal Length, Sepal Width, Petal Length, Petal Width
  - Output: Predicted species with confidence score

- 📸 **Image-Based Prediction**
  - Upload flower image
  - Uses PlantNet API for classification

- 🌍 **Multilingual Support**
  - Translate website into regional languages
  - Helpful for non-English users

- 📖 **Dynamic Species Information**
  - Displays habitat, growing conditions, and features after prediction

---

## 🧠 Machine Learning Models Used

- Logistic Regression  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)  
- Decision Tree  
- Neural Network (MLP)  

👉 Best model selected based on accuracy

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **ML Library:** Scikit-learn  
- **API:** PlantNet API
